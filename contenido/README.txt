CARPETA DE CONTENIDO
====================

Esta carpeta contiene todos los archivos de tu portafolio.

ESTRUCTURA:
-----------
contenido/
├── tareas/
│   └── tarea-001/
│       ├── documento.pdf    (archivo original)
│       ├── pagina-1.jpg     (imagen de pagina 1)
│       └── pagina-2.jpg     (imagen de pagina 2)
├── laboratorios/
│   └── lab-001/
│       └── ...
├── parciales/
│   └── parcial-001/
│       └── ...
└── reflexiones/
    └── (imagenes opcionales)

COMO AGREGAR CONTENIDO:
-----------------------
1. Crea una carpeta con el ID del item (ej: tarea-002)
2. Coloca el archivo original (PDF) como "documento.pdf"
3. Convierte las paginas a imagenes JPG
4. Nombra las imagenes como "pagina-1.jpg", "pagina-2.jpg", etc.
5. Actualiza el archivo JSON correspondiente en data/

CONVERTIR PDF A IMAGENES:
-------------------------
Usa https://www.ilovepdf.com/pdf_to_jpg para convertir tus PDFs a imagenes facilmente.
